## 🎯 About

TomatoGuard is a comprehensive research project developed at Sri Lanka Institute of Information Technology (SLIIT) that revolutionizes tomato farming through:

- **AI-Powered Disease Detection** - 95% accuracy using CNN models
- **IoT Sensor Integration** - Real-time environmental monitoring
- **Mobile Application** - User-friendly interface for farmers
- **Community Platform** - Knowledge sharing among farmers
- **Harvest Prediction** - LSTM models for optimal timing

## ✨ Features

### 🔬 Core Technologies
- **Disease Detection**: Advanced CNN models (VGG16, YOLO) for identifying plant diseases
- **Pest Identification**: Smart detection system for early intervention
- **Environmental Monitoring**: ESP32-based sensor network
- **Weather Integration**: Real-time weather data and forecasting
- **Harvest Prediction**: LSTM models for optimal harvest timing

### 📱 Mobile Application
- Real-time disease scanning
- IoT dashboard with sensor data
- Alert notifications system
- Community forum
- Fertilizer recommendations
- Offline capability

### 🌐 Web Platform
- Project documentation
- Research findings
- Download resources
- Team information
- Contact system

## 🛠 Technology Stack

### Frontend
- **React** - Modern UI library
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Lucide React** - Beautiful icons
- **Vite** - Fast build tool

### Backend & AI
- **Python** - Core backend development
- **TensorFlow/Keras** - Deep learning models
- **OpenCV** - Image processing
- **Flask/FastAPI** - API development
- **SQLite/PostgreSQL** - Database

### IoT & Hardware
- **ESP32** - Microcontroller platform
- **Sensors** - Temperature, humidity, soil moisture
- **Arduino IDE** - Hardware programming
- **MQTT** - IoT communication protocol

### Deployment
- **Netlify** - Frontend hosting
- **Heroku/AWS** - Backend deployment
- **GitHub Actions** - CI/CD pipeline

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Git

### Installation

1. **Clone the repository**

git clone https://github.com/yourusername/tomato-guard.git
cd tomato-guard


2. **Install dependencies**
npm install

or
yarn install


3. **Start development server**
npm run dev

or
yarn dev


4. **Open your browser**
http://localhost:5173


### Build for Production
npm run build

or
yarn build
